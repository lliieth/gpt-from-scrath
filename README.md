# CyberSec GPT Project

This project implements a simplified GPT-style language model using PyTorch.

The model is trained on cybersecurity-related text and demonstrates how a Large Language Model (LLM) works internally.

## Features

- Transformer Architecture
- Self-Attention Mechanism
- Pretraining and Fine-tuning
- Cybersecurity domain dataset
- Text generation demo

## Project Files

attention.py → Self-attention implementation

transformer.py → Transformer architecture

pretrain.py → Model pretraining

finetune.py → Fine-tuning for cybersecurity data

CyberSec_GPT_Project.ipynb → Google Colab demo

cyber_dataset.txt → Cybersecurity dataset

## Cybersecurity Topics

- Malware
- Phishing
- Ransomware
- Vulnerabilities
- Encryption
- Network Security

## Goal

To demonstrate how a simplified GPT model can be built from scratch and adapted to the cybersecurity domain.
